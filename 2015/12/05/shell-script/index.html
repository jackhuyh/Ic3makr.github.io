Shell Script

发表于 2015-12-05   |   分类于 Linux   |  

学习shell script的过程中的一部分知识点整理

find

find ~ -type f -name 'foo*' -exec ls -l '{}' + 
find ~ -type f -name 'foo*' -print | xargs ls -l

解释1：寻找主目录下文件名为 foo 开头的文件，在全部找到后执行

ls -l

解释2：寻找主目录下文件名为 foo 开头的文件，然后将输出结果通过管道传到xargs的标准输入

xargs

xargs 命令能够将标准输入进行分割生成一个参数列表，例如：

➜  ~  xargs
a b c
a b
c
a b c a b c

xargs 在读取标准输入的资料时，会自动忽略空白行，多余的空白或者是tab也会自动忽略，想要xargs执行一个指令，只需将其跟在后面即可

➜  ~  xargs cat
arg1 arg2
arg3

相当于是

➜  ~  cat arg1 arg2 arg3

通过-n选项可指定接收的参数上限，-p选项进行命令执行前的确认

xargs与find结合

find . -name '*.c' | xargs grep 'stdio.h'

表示寻找到当前目下的所有.c文件，并将这些文件处理为参数列表，作为rm命令的参数

xargs与grep结合

➜  alice git:(master)find . -name "*.c" | xargs -p rm -f
./alice.c:#include <stdlib.h>
./linkcap.c:#include <stdlib.h>

➜  alice git:(master)grep 'stdlib.h' alice.c linkcap.c
./alice.c:#include <stdlib.h>
./linkcap.c:#include <stdlib.h>

上面两条命令等效，意思很简单，不解释了

实例

➜  ~  md playground/dir-{00{1..9},0{10..99},100}
➜  ~  touch playground/dir-{00{1..9},0{10..99},100}/    file-{A..Z}
➜  ~  find ~/playground -name "file-E" -type f| xargs -I {} mv {} {}/../FILE-E

搭建实验环境

find playground -type f -name 'file-A'

寻找playground目录下所有的文件名为 file-A 的文件
